#To Do List in React JS

## How to start

### Fist step
inside the 'backend' folder run this line in the terminal

- yarn
- yarn dev

or

- npm install
- npm run dev

### Second step
inside the 'react-front' folder run this line in the terminal

- yarn 
- yarn start

or 

- npm install 
- npm run start

### Finally

access in your browser this URL --> http://localhost:3000
